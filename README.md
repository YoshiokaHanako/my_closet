# my closet

## サイト概要
### サイトテーマ
お気に入りのファッションアイテムや購入したアイテムを1つずつアップできる投稿型アプリ。
​
### テーマを選んだ理由
私はアパレル会社で働いており、日々多くのファッションアイテムに触れる中で、自分のお気に入りや最近購入したアイテムを整理して振り返る方法が欲しいと感じていました。
現状は購入履歴やコーディネートの参考を個人で管理するのが難しく、SNSやスマホのメモだけでは整理が不十分で、過去に買ったアイテムを見返すのに時間がかかってしまうこともあります。
また、他のファッション好きの人にとっても、買い物の参考になる情報や、コーディネート例を気軽に共有・閲覧できる場所が少ないと感じています。
そこで、自分だけでなく他のユーザーも簡単にアイテムを投稿・管理できる仕組みを作ることで、買い物の参考になり、自分のクローゼットの中身を見返せる便利なツールを提供できると考え、このポートフォリオ制作のテーマに選びました。
​
### ターゲットユーザ
ファッションが好きで、自分の購入アイテムを整理したい人
購入前に他の人のアイテム投稿を参考にしたい人
クローゼットの中身を簡単に振り返りたい人
​
### 主な利用シーン
作品を閲覧し、気に入った商品を購入したい時
SNSで見た作品を詳しく見たい時
特別な贈り物を探している時
